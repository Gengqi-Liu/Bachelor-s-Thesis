\begin{abstract}
This thesis presents the design, implementation, and experimental evaluation of an audio-based MIMO-OFDM demonstrator intended for education-oriented analysis of physical-layer communication principles. Building on an existing software framework, the system is adapted to a modern multi-channel audio interface, restructured using MATLAB App Designer, and extended with a transparent and configurable receiver-side signal-processing chain.

The demonstrator enables end-to-end transmission of text payloads over an acoustic multi-input multi-output channel using spatial multiplexing and OFDM modulation. Synchronous multi-channel audio playback and recording are employed to ensure reproducible operation. At the same time, receiver processing includes timing synchronization based on a Schmidl--Cox preamble, training-based channel estimation using a least-squares / zero-forcing approach, and per-subcarrier linear MMSE detection.

Beyond payload reconstruction, the system exposes a comprehensive set of physical-layer observables, including synchronization metrics, estimated channel impulse and transfer functions, equalized symbol constellations, channel rank and conditioning measures, and error vector magnitude (EVM). These quantities enable detailed inspection of the interaction between channel characteristics, receiver algorithms, and symbol-level performance in a real acoustic MIMO environment.

Experimental results obtained with a $4\times8$ spatial multiplexing configuration using 4-QAM modulation demonstrate stable and repeatable operation with reliable payload recovery, despite non-ideal constellation quality under certain parameter settings. The developed demonstrator thus provides a structured and extensible platform that bridges communication theory and hands-on experimentation, and is well suited for both teaching and exploratory analysis of MIMO-OFDM systems.
\end{abstract}
